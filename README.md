## ✨ المميزات

### 🔤 دعم OpenType Features الكامل
- **Ligatures** - الربط (liga, dlig, clig, rlig)
- **Alternates** - البدائل (calt, salt, swsh)
- **Stylistic Sets** - المجموعات الأسلوبية (ss01-ss20)
- **Arabic Features** - الخصائص العربية (init, medi, fina, isol)
- **Numeric Features** - الأرقام (lnum, onum, tnum, frac)
- **+80 خاصية أخرى...**

### 🎚️ Variable Fonts
- دعم كامل للخطوط المتغيرة
- التحكم في المحاور (Weight, Width, Slant, etc.)
- معاينة مباشرة للتغييرات

### 🖼️ محرر احترافي
- طبقات متعددة (نصوص، صور، أشكال)
- تحكم كامل في التحويلات
- تأثيرات النص (ظل، حدود، تدرج)
- تصدير بجودة عالية

### 🌍 دعم اللغة العربية
- واجهة عربية بالكامل
- دعم RTL
- خطوط عربية مدمجة
